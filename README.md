# 😂 Jokes & Cat Facts API Web Application

A web application that integrates public APIs to retrieve and display random jokes and cat facts. The application allows users to interact with external APIs through GET requests and submit their own jokes using POST requests.

This project was created as part of an API Integration project requirement.

---

# 🚀 Features

- Fetch random jokes from JokeAPI
- Retrieve random cat facts from Cat Facts API
- Submit custom jokes through POST request
- User-friendly and responsive interface
- Real-time API data retrieval
- Error handling for failed API requests
- Loading states during API requests
